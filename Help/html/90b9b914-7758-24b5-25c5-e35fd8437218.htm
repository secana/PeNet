PeNet Namespace
This is the main namespace where you find all functions and classes to analyse a PE file and interpret the different values and flags.
Classes
  ClassDescription
Constants
This class contains constants and flags which are used in a PE file. The constants can be used to map a numeric value to an understanable string.
PeFile
This class represents a Portable Executable (PE) file and makes the different header and properites accessable.
PeFileCrlUrlList
This class parses the Certificate Revocation Lists of a signing certificate. It provides access to all CRL URLs in the certificate.
PeFileExportFunction
Represents an exported function.
PeFileImportFunction
Represents an imported function.
Utility
This class provides useful functions to work with PE files.
UtilityFileCharacteristics
Describes which file characteristics based on the file header are set. The ToString Method creates a readable string containing all the information.
